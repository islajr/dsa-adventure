# LC 226: Invert Binary Tree

> **Description:** [LC 226](https://leetcode.com/problems/invert-binary-tree/description/)
>
> **Difficulty:** Easy
>
> **Time Complexity:** O(n)
>
> **Space Complexity:** O(n) -- Recursive stack

## Approach

This approach uses Depth-first search to traverse the tree, swapping the children at every node, and then recursively inverting the rest of the sub-tree side-by-side.
